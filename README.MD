### Rust book web server

This repository contains implementation of the Final Project from [The Rust Book](https://doc.rust-lang.org/book/)

## How to build
1. Install Rust
2. Clone project
3. Build project
```
cd rust_web_server
cargo build
```

## How to run

1. Run executable
```
cargo run
```
2. Open browser and follow to `127.0.0.1:7878`.
